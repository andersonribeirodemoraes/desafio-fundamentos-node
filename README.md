## Instalação e rodando o projeto
```sh
$ git clone https://github.com/andersonribeirodemoraes/desafio-fundamentos-node.git
$ cd desafio-fundamentos-node  
$ yarn
$ yarn dev:server

Rodando em: http://localhost:3333
```

## Rodando os testes
```sh
$ yarn test
```
